Qu'est-ce-que NumPy?

  • NumPy est une librairie utilise dans presque tous les projets de calcul numrique sous Python
  • NumPy fournit des structures de donnes performantes pour la manipulation de vecteurs, de matrices, de tenseurs. On parlera de tableau ou array en Anglais
  • NumPy est crit en C et en Fortran d'o ses performances leves lorsque les calculs sont vectoriss (formuls comme des oprations sur des tableaux)

Qu'est-ce-que matplotlib?

  • matplotlib est une librairie pour la gnration de graphiques en 2D et 3D
    • syntaxe trs proche de celle de Matlab
    • supporte texte et tiquettes en LaTeX
    • sortie de qualit dans divers formats (PNG, PDF, SVG, EPS...)
    • interface graphique intractive pour explorer les figures

Pour utiliser NumPy et matplotlib il faut commencer par les importer :
